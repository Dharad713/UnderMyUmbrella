# UnderMyUmbrella
I want to make a service which texts my phone about if/when it will rain throughout the day, so I know to bring an umbrella. 
This service is not novel, but is being made mostly as a learning experience. This was done in a day and is only designed for personal use


This uses textbelt's api, which requires a purchase of texts. This is the easiest/cheapest way for me to send specifically text messages, but this could be modified to use an email based system

## CLI args:
- `-D` or `--dry` : run the program without sending a text
